# Domain elements

Click **Save**{.uicontrol} to save the file.

The **Dashboard**{.wintitle} window opens.

Navigate to **File > Open > Recent**{.menucascade} to see recent files.

Edit the `config.yaml`{.filepath} file.

Run the `ansible-playbook`{.cmdname} command.

Enter `admin`{.userinput} as the username.

The system returns `OK`{.systemoutput}.

Set the `verbose`{.option} option.

Use the `inventory`{.parmname} parameter.

Call the `getHosts`{.apiname} API.

Set the `MAX_RETRIES`{.varname} variable.

Check the `ERR_TIMEOUT`{.msgph} message.

Use `codeph`{.codeph} for code phrases.

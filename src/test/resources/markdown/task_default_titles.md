# Install the package

This topic describes how to install the package.

## Prerequisites

- A supported operating system.
- Administrator access.

## About this task

This procedure walks you through a standard installation.

1.  Download the installer.

    Save the file to your local drive.

1.  Run the installer.

## Verification

Check the installation was successful:

1.  Run the version command.
